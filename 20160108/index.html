$ sudo npm install ungit -g

Dans Sublimetext, installer avec Package Control : git & gitgutter

Créer des branch à l'aide de git ou de ungit
Création d'une branche : git checkout -b `nomdelabranche`
Déplacement sur une branche : git checkout 'nomdelabranche'
Merger une branche : se mettre sur la branche non modifiée puis git merge 'nomdelabranche'
Supprimer une banche : git branch -d 'nomdelabranche' ou git branch -D 'nomdelabranche'
